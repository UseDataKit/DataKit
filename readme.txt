Displays Gravity Forms entries in a table using the native WordPress DataViews component.

== Description ==

This plugin harnesses the power of [WordPress DataViews](https://developer.wordpress.org/block-editor/reference-guides/packages/packages-dataviews/) to display data.

Read more about the [DataKit plugin](https://www.datakit.org/plugin/).

== Installation ==

1. Upload plugin files to your plugins folder, or install using WordPress's built-in Add New Plugin installer
2. Activate the plugin
